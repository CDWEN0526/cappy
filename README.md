# Capybara Shell

#### 介绍
类似于finalshell，xshell之类的工具，ssh远程服务器管理工具，“卡皮巴拉shell”。

😁由于本人是运维仔，并且初尝Golang，写的不好多见谅，如有问题可以给我提issues，我有空就会看一下。

🤡目前只有windows版本，至于linux和mac版本后续会推出，目前没有mac电脑所以打包比较难。
