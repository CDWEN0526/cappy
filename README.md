## 项目简介

Cappy（卡皮巴拉 Shell）是一款功能强大的终端工具，类似于 FinalShell、XShell 等知名软件。作为一名运维工程师利用业余时间开发的个人项目，Cappy 致力于为运维工作提供更加便捷高效的体验。

### 设计理念

- **简单易用** - 直观界面设计，快速上手
- **高效管理** - 批量操作、多屏协作提升效率
- **安全可靠** - 加密传输保障数据安全
- **持续迭代** - 根据实际需求不断优化更新

## 核心功能

### 🔐 SSH 远程连接

支持多服务器统一管理，提供快速稳定的 SSH 连接体验。

- **多服务器管理** - 统一的服务器列表管理，支持分组
- **连接会话** - 支持多个会话并行连接，互不干扰
- **自动重连** - 网络中断后自动尝试重连
- **密钥认证** - 支持密码和密钥两种认证方式

### 📁 SFTP 文件传输

内置 SFTP 客户端，实现便捷的远程文件管理。

- **双向传输** - 支持上传和下载文件
- **目录操作** - 创建、删除、重命名目录
- **文件预览** - 支持文本常见文件格式预览
- **拖拽操作** - 支持拖拽方式上传文件

### 🚀 Trzsz 传输

集成 Trzsz（rz/sz 的现代替代方案），提供更高效的文件传输体验。

- **大文件传输** - 优化的传输算法，支持大文件快速传输
- **断点续传** - 传输中断后可继续传输
- **进度显示** - 实时显示传输进度

## 效率工具

### ⚡ 快捷指令

自定义常用命令，一键执行，提升日常运维效率。

- **命令分组** - 将相关命令分组管理
- **参数变量** - 支持自定义参数，灵活执行
- **快捷键绑定** - 为常用命令绑定快捷键

### 🖥️ 分屏操作

支持多终端同时显示，方便对比和同时操作多个服务器。

- **水平/垂直分屏** - 灵活的分屏模式
- **多会话管理** - 同时管理多个终端会话
- **同步操作** - 可配置命令广播功能

### 📡 命令广播

向多个服务器同时发送相同命令，实现批量操作。

- **批量执行** - 一次操作管理多台服务器
- **结果对比** - 可对比不同服务器的返回结果
- **错误提示** - 快速定位执行失败的服务器

### 📊 监控仪表盘

实时监控系统资源使用情况，异常情况及时告警。

- **CPU 监控** - 实时显示 CPU 使用率
- **内存监控** - 监控内存使用情况
- **磁盘监控** - 磁盘空间和使用率监控
- **网络监控** - 网络流量监控

## 高级特性

### 🤖 AI Agent

集成 OpenAI 能力，提供智能运维辅助。

- **智能问答** - 回答运维相关问题
- **命令建议** - 根据场景推荐合适的命令
- **脚本生成** - 自动生成运维脚本
- **错误诊断** - 分析错误信息并提供解决方案
- **智能体操控** - ai接管服务器控制

### 🌐 AI 翻译

内置智能翻译功能，方便阅读英文文档。

- **多语言支持** - 支持多种语言互译

### 🔗 内网穿透

提供便捷的内外网访问解决方案。

- **快速配置** - 简单配置即可实现内网穿透

### ☁️ WebDAV 同步

配置数据云端同步，多设备无缝切换。

- **自动同步** - 配置变更自动同步
- **多设备同步** - 在不同设备上使用相同配置

### 🔍 进程网络监控

实时监控系统进程和网络状态。

- **网络连接** - 监控网络连接状态
- **进程连接** - 详细检测到每个进程的连接数，连接带宽

### 🎨 背景图设置

支持自定义界面背景图，打造个性化工作环境。

- **自定义背景** - 设置本地或在线图片作为背景
- **透明度调节** - 调节背景透明度
- **主题切换** - 支持深色/浅色主题切换

## 技术架构

| 层级 | 技术选型 | 说明 |
|------|----------|------|
| **后端框架** | Golang + Wails | 提供原生性能表现和跨平台 GUI 支持 |
| **前端框架** | Vue.js 3 + Element Plus | 构建现代化、响应式的用户界面 |
| **状态管理** | Pinia | Vue 3 官方推荐的状态管理库 |
| **构建工具** | Vite | 极速的现代前端构建工具 |
| **终端模拟** | xterm.js | 强大的 Web 终端模拟器 |
| **通信协议** | SSH2 + SFTP | 标准安全的远程连接协议 |
| **文件传输** | Trzsz | 高效的终端文件传输工具 |

## 常见问题

**Q: 为什么只能 Windows 版本？**

A: 作为个人业余项目，开发者时间和精力有限。Windows 是目前运维工作中使用最广泛的系统，因此优先开发 Windows 版本。

**Q: 支持多标签页吗？**

A: 是的，Cappy 支持多标签页和多窗口模式。

**Q: AI 功能如何使用？**

A: 在设置中配置 OpenAI API Key 即可使用 AI 功能。
